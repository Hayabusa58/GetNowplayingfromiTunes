# お読みください

## 動作確認環境
- .NetFrameWork v3.5
- iTunes v12.6.0.100

## 機能概要
- 「get info」ボタンをクリックすると再生中の曲名、歌手名、アルバム名を取得してウィンドウに表示します
- 「copy to clipboard」ボタンをクリックすると取得した曲情報を「『#nowplaying』 + 『/』 + 『アーティスト名』」の形式でコピーします
